### На основе анализа графика, процедура V1 работает быстрее процедуры V2.

1. **Более высокая пропускная способность у процедуры V1**
2. **У процедуры V1 лучше производительность** — процедура V1 обрабатывает почти в 2 раза больше запросов в секунду в пике, чем процедура процедура V2
3. **Более стабильные показатели** — процедура V1 показывает значения в диапазоне 2,000-5,000 hits/sec
4. **Процедура V2 имеет более широкий разброс** — 300-2,700 hits/sec.
